Sobre mim

Atualmente trabalho como Assistente de Desenvolvimento de Software na AMEnergia, uma empresa de soluções para o gerenciamento de energia elétrica, visando a redução dos custos.

Sou responsável pela extração e inserção dos dados em grande escala das faturas de energia elétrica. Para isso, utilizo o desenvolvimento de scripts em PHP e softwares para automação da extração de dados de diversas páginas Web. Na programação, essas técnicas são conhecidas como Web Scraping e Web Automation. Através desses processos, conseguimos cadastrar faturas para grandes clientes, como a Claro S.A, que possui milhões de unidades no Brasil inteiro.
